项目简介Introduction

Lumen 是一个自托管的 AI 网关与对话管理平台，基于 Python Flask 构建。支持 9 家内置 AI 厂商（DeepSeek、OpenAI、Anthropic、Kimi、火山引擎、阿里云、xAI、智谱、Minimax）以及自定义 OpenAI 兼容端点。提供多用户系统、细粒度权限控制、接口开关管理和 简洁的管理后台。

Lumen is a self-hosted AI gateway and conversation management platform built with Python Flask. It supports 9 built-in AI vendors (DeepSeek, OpenAI, Anthropic, Kimi, Volcano Engine, Alibaba Cloud, xAI, Zhipu, Minimax) plus custom OpenAI-compatible endpoints. Features include multi-user system, fine-grained permission control, endpoint toggle management, and an clean admin dashboard.

权限系统Permissions

Lumen 采用两级权限控制：全局默认权限 + 用户单独权限。用户单独权限可设为"继承全局"或"自定义覆盖"。权限项包括：每日对话次数、可用模型、单次最大 Token、导出开关、文件上传、频率限制。

Lumen uses a two-tier permission system: global defaults + user-specific overrides. User permissions can be set to "inherit global" or "custom override". Permission items include: daily chat limit, allowed models, max tokens per request, export toggle, file upload, rate limiting.

接口开关Endpoint Toggles

系统支持按分组或单个接口控制 API 的启用/禁用。关闭的接口返回 403。管理后台接口（/api/admin/*）始终可用，不受开关影响。分为 3 个分组：认证(auth)、对话(chat)、管理后台(admin)。

The system supports enabling/disabling APIs by group or individual endpoint. Disabled endpoints return 403. Admin endpoints (/api/admin/*) are always available regardless of toggle state. 3 groups: auth, chat, admin.

完整插件模板Complete Plugin Template

# my_plugin.py
from plugins import LumenPlugin, PluginMeta, ToolDefinition, PageDefinition, ApiRouteDefinition
from flask import jsonify, request
from models import db
import sqlalchemy as sa
from typing import Any, Dict, List

PLUGIN_META = {
    "name": "my_plugin",           # 唯一标识（文件名）
    "display_name": "My Plugin",   # 显示名称
    "version": "1.0.0",            # 语义化版本
    "description": "A comprehensive example plugin",
    "author": "Your Name",
    "priority": 50,                # 0-100, 越大越先执行
    "category": "tool",            # tool / utility / integration / content
}

def create_plugin(meta: PluginMeta) -> "MyPlugin":
    return MyPlugin(meta)

# ── 可选：自定义数据库模型 ──
class MyData(db.Model):
    __tablename__ = "my_plugin_data"
    id = sa.Column(sa.Integer, primary_key=True)
    key = sa.Column(sa.String(255), unique=True)
    value = sa.Column(sa.Text)

class MyPlugin(LumenPlugin):
    # ══════ AI 工具 ══════
    def get_tools(self):
        return [
            ToolDefinition(
                name="my_tool",
                description="Do something useful with input text",
                parameters={
                    "type": "object",
                    "properties": {
                        "input": {"type": "string", "description": "Input text to process"},
                        "mode": {"type": "string", "enum": ["fast", "deep"], "default": "fast"}
                    },
                    "required": ["input"]
                }
            ),
            ToolDefinition(
                name="my_data_lookup",
                description="Look up data stored by this plugin",
                parameters={
                    "type": "object",
                    "properties": {
                        "key": {"type": "string", "description": "Data key to look up"}
                    },
                    "required": ["key"]
                }
            ),
        ]

    def execute_tool(self, tool_name, arguments, ctx):
        if tool_name == "my_tool":
            mode = arguments.get("mode", "fast")
            result = f"[{mode}] Processed: {arguments['input']}"
            # 保存到数据库
            entry = MyData(key=f"tool_{arguments['input'][:20]}", value=result)
            db.session.add(entry)
            db.session.commit()
            return {"result": result, "id": entry.id}
        if tool_name == "my_data_lookup":
            entry = MyData.query.filter_by(key=arguments["key"]).first()
            return {"found": bool(entry), "value": entry.value if entry else None}
        return {"error": f"Unknown tool: {tool_name}"}

    # ══════ 消息钩子 ══════
    def pre_chat(self, ctx):
        """消息发送前：可修改 content、system_prompt、tools、model"""
        # 添加系统指令
        if "system_extra" not in ctx:
            ctx["system_extra"] = ""
        ctx["system_extra"] += "\n[MyPlugin: active]"
        # 动态注入额外工具
        extra_tools = self.get_tools()
        ctx.setdefault("tools", []).extend(extra_tools)
        return ctx

    def post_chat(self, response, ctx):
        """AI 响应后：可修改响应文本"""
        if ctx.get("model") and "gpt" in ctx["model"]:
            return response + "\n\n---\n*Enhanced by MyPlugin v" + self.meta.version + "*"
        return response

    def on_error(self, error, ctx):
        """错误处理：返回降级响应或 None（继续抛出）"""
        if "timeout" in str(error).lower():
            return "Sorry, the request timed out. Please try again."
        return None  # 让其他插件或系统处理

    # ══════ 自定义页面 ══════
    def register_pages(self):
        return [
            PageDefinition(
                page_id="my_dashboard",
                title="My Dashboard",
                icon="layout-dashboard",
                html_content="""

  
Plugin Dashboard

  

    

      
-

      
Records

    

  

  

  

"""
            )
        ]

    # ══════ 自定义 API ══════
    def register_api_routes(self):
        def get_data():
            entries = MyData.query.order_by(MyData.id.desc()).limit(50).all()
            return jsonify({
                "count": len(entries),
                "items": [{"id": e.id, "key": e.key, "value": e.value} for e in entries]
            })

        def save_data():
            data = request.get_json() or {}
            entry = MyData(key=data.get("key"), value=data.get("value", ""))
            db.session.add(entry)
            db.session.commit()
            return jsonify({"success": True, "id": entry.id})

        def delete_data(key):
            MyData.query.filter_by(key=key).delete()
            db.session.commit()
            return jsonify({"success": True})

        return [
            ApiRouteDefinition(method="GET", path="/data", handler=get_data, auth_required=False),
            ApiRouteDefinition(method="POST", path="/data", handler=save_data, auth_required=True),
            ApiRouteDefinition(method="DELETE", path="/data/", handler=delete_data, auth_required=True),
        ]

    # ══════ 数据库表 ══════
    def register_db_tables(self):
        return [MyData]

    # ══════ 中间件 ══════
    def register_middleware(self):
        def log_request():
            from flask import g
            import time
            g.request_start = time.time()

        def log_response(response):
            from flask import g
            import time
            elapsed = time.time() - getattr(g, 'request_start', time.time())
            response.headers['X-Plugin-MyPlugin'] = f'{elapsed:.3f}s'
            return response

        return [
            {"type": "before_request", "handler": log_request, "priority": 10},
            {"type": "after_request", "handler": log_response, "priority": 10},
        ]

    # ══════ 事件系统 ══════
    def register_events(self):
        def on_message_sent(data):
            """当用户发送消息时触发"""
            user = data.get("user", "unknown")
            model = data.get("model", "unknown")
            # 可以记录统计、发送通知等
            print(f"[MyPlugin] Message sent by {user} using {model}")

        def on_system_startup(data):
            """系统启动时触发"""
            print("[MyPlugin] System started, initializing...")

        return {
            "message.sent": on_message_sent,
            "system.startup": on_system_startup,
        }

    # ══════ 定时任务 ══════
    def register_cron_jobs(self):
        def cleanup_old_data():
            """每小时清理超过 7 天的数据"""
            from datetime import datetime, timedelta
            cutoff = datetime.utcnow() - timedelta(days=7)
            MyData.query.filter(MyData.created_at < cutoff).delete()
            db.session.commit()

        return [
            {"name": "cleanup_old_data", "interval_seconds": 3600, "handler": cleanup_old_data, "enabled": True}
        ]

    # ══════ 生命周期 ══════
    def on_install(self):
        """首次安装时创建数据库表"""
        db.create_all()
        return True

    def on_uninstall(self):
        """卸载时清理数据"""
        MyData.query.delete()
        db.session.commit()
        return True

    def on_upgrade(self, from_version, to_version):
        """升级时迁移数据"""
        if from_version.startswith("0."):
            # v0.x → v1.x 迁移逻辑
            pass
        return True

    def on_configure(self, config):
        """配置变更时调用"""
        self.config = config
        return True

数据库配置Database Configuration

Lumen 支持三种数据库后端：SQLite（默认，零配置）、PostgreSQL、MySQL。配置方式包括管理后台、新手引导或环境变量。修改数据库类型后需重启服务生效。

Lumen supports three database backends: SQLite (default, zero-config), PostgreSQL, and MySQL. Configure via admin panel, onboarding wizard, or environment variables. Server restart required after changing database type.

环境变量Env Variable说明Description默认值Default
DB_TYPE数据库类型Database typesqlite
DB_HOST数据库主机Database hostlocalhost
DB_PORT数据库端口Database port5432 / 3306
DB_NAME数据库名Database nameapp.db / lumen
DB_USER数据库用户Database user—
DB_PASSWORD数据库密码Database password—

数据库配置 APIDatabase Config API

GET /api/admin/config/database

获取当前数据库配置（密码已隐藏）Get current database config (password masked)

PUT /api/admin/config/database

更新数据库配置（需重启）Update database config (restart required)

POST /api/admin/config/database/test

测试数据库连接Test database connection

系统配置System Config

系统配置以 key-value 形式存储，支持加密字段（如 API Key）。可配置项包括：AI 厂商/API Key/模型、开放注册、管理员查看对话权限、对话保留天数等。

System config is stored as key-value pairs, with encryption support for sensitive fields (e.g., API Key). Configurable items include: AI vendor/API key/model, open registration, admin view content permission, conversation retention days, etc.

部署Deployment

推荐使用 Docker 部署，或直接通过 Python 运行。生产环境建议使用 PostgreSQL 替代 SQLite，并通过环境变量配置数据库连接、JWT 密钥等。

Docker deployment is recommended, or run directly with Python. For production, use PostgreSQL instead of SQLite and configure database connection and JWT secret via environment variables.
